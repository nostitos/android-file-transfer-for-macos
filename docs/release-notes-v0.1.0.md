# Android File Transfer for macOS v0.1.0

The first public pre-release of a free and open-source USB file-transfer utility for Android devices and macOS.

## Included

- Separate Apple-silicon and Intel DMGs for macOS 12 or newer.
- Phone and Mac file panes with queued copies in both directions.
- Folder transfer, Finder drag and drop, progress, cancellation, retry, and conflict-safe naming.
- Plain-language USB, File Transfer, file-session, and permission diagnostics.
- No account, cloud service, subscription, companion Android app, analytics, or telemetry.

Both DMGs and their contained apps are Developer ID signed, Apple-notarized, stapled, and independently checked after upload. Verify downloads with `SHA256SUMS.txt`.

This is an early public release. Real-device validation has focused on Samsung MTP behavior; reports from other Android devices and macOS versions are especially useful.
